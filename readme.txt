Prerequisites:
1) The following perl modules are to be installed from CPAN
		a) DBD::MySQL
		b) DBI
		c) Net::SNMP
		d) RRD::Simple
		e) Data::Dumper
		f) LWP::Simple


Steps to run:
1) Replace db.conf file with your own file or change the credentials respectively.
2) Move to the current directory through terminal
3) Give necessary permissions to the folder.
4) Run backend.sh file using terminal prompt.
5) Now, go to the local web browser and type localhost/et2536-vegr15/assignment2/index.php.
